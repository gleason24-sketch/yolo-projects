# Lead Outreach Email Generator

## What It Does
Generates personalized cold outreach emails for Riot Digital leads.

## Usage

```bash
node generate-email.cjs --company "Plumbing Company" --industry "plumbing" --pain "booking calls"
```

## Output
Creates a personalized email ready to send.
